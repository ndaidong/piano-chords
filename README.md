# piano-chords
Simple tool for determining piano chords based on given scale.


## Install & Build

```
git clone https://github.com/techpush/piano-chords.git
cd piano-chords
npm install
npm start

// build
npm run build

// test
npm test

// e2e test
// ensure that node process is running with "npm start"
// then in another tab
npm run testcafe
```
